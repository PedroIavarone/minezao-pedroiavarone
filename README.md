# 🎮 Minezão do Pedro

EaglercraftX 1.8 hospedado online — jogue Minecraft no navegador sem precisar instalar nada.

## 🚀 Como jogar

Acesse diretamente pelo link:

👉 **[minezao-pedroiavarone.vercel.app](https://minezao-pedroiavarone.vercel.app)**

Nenhuma instalação necessária. Funciona direto no navegador.

## 🖥️ Requisitos

- Navegador moderno (Chrome recomendado)
- Conexão com a internet

## 📦 Sobre o projeto

Este repositório hospeda o **EaglercraftX 1.8.8 Offline**, versão criada por **lax1dude**.  
O jogo é servido via **Vercel** para acesso rápido e fácil.

> EaglercraftX é um projeto de código aberto que permite jogar Minecraft 1.8 no navegador via WebAssembly.
